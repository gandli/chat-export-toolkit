# Chat Export Toolkit

[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/licenses/MIT)
[![Version](https://img.shields.io/badge/version-0.5.1-01B259.svg)](CHANGELOG.md)

A Tampermonkey userscript specifically designed for **Tencent Yuanbao**. It perfectly captures conversation history and supports exporting it into structured Markdown, JSON, or DOCX files.

---

## ✨ Core Features

- **🚀 Zero-config Export**: Export current or all conversations with one click, no complex setup required.
- **📄 Multi-format Support**:
  - **Markdown**: Perfectly adapted for knowledge bases like Obsidian and Notion.
  - **JSON**: Convenient for developers to perform structured data processing.
  - **DOCX**: Native Word documents generated by a built-in engine, suitable for sharing and printing.
- **📦 Batch Export (ZIP)**: Automatically fetch all historical conversations and package them into a ZIP file, so you never lose any inspiration.
- **⚡️ Rapid Response**:
  - Conversations are silently pre-loaded in the background—no waiting for export.
  - One-click quick copy of MD/JSON to the clipboard.
- **🎨 Exqusite UI**: A UI system that deeply integrates with Yuanbao's native design style, providing an immersive interactive experience.

## 🛠 Installation Guide

1. **Environment Setup**: Ensure your browser has the [Tampermonkey](https://www.tampermonkey.net/) extension installed.
2. **Download Script**: Open `userscripts/chat-export.user.js` in this repository and click the **Raw** button to trigger installation.
3. **Get Started**: Visit the [Tencent Yuanbao Official Website](https://yuanbao.tencent.com/), and you will see the "Export" float panel in the bottom right corner.

## 💡 Technical Implementation

- **Request Interception**: Based on XMLHttpRequest/Fetch hooks, it intercepts conversation packets in real-time, completely solving the issue of content truncation.
- **DOM Fallback**: Even if API requests are restricted, it can intelligently parse and grasp visible content from the page via the DOM.
- **Modern Styling**: Uses native CSS variables and micro-animations to ensure the script remains lightweight while maintaining a high-end visual feel.
